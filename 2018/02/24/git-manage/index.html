git stash 使用

暂存stash

将当前修改暂存到stash中, message可用去记录版本

查看所有stash

提取stash

git stash pop将获取暂存中的第一条记录

git stash apply stash@{index}获取暂存中对应索引的暂存记录

移除stash

git stash drop stash@{index}删除指定索引的记录

git stash clear 删除所有stash记录

查看对应stash修改信息

git stash show 查看最近一条记录的暂存文件

git stash show -p 查看最近一条记录的暂存文件的修改内容

git stash show stash@{index} -p 查看指定记录的暂存文件的修改内容

从stash检出分支

git stash branch <branch-name> 从当前暂存的记录下,检出新的分支,暂存记录将被删除

git stash branch <branch-name> [<stash@{index}>] 从指定索引的暂存记录,检出新的分支,暂存记录将被删除
